# dayofyear()

Returns the integer number represents the day number of the given year.

```kusto
dayofyear(datetime(2015-12-14))
```

## Syntax

`dayofyear(`*a_date*`)`

## Arguments

* `a_date`: A `datetime`.

## Returns

`day number` of the given year.
